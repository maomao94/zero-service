# 模板说明

本目录提供新项目开发流程的标准模板文件。
创建新项目时，将本目录下的模板文件复制到以业务名称命名的新目录中。

## 使用方法

```bash
# 示例：创建「DJI上云前置服务」项目目录
cp -r CP-开发流程/template/ CP-开发流程/DJI上云前置服务/
```

复制后，将各文件中的 `{项目名称}` 等占位符替换为实际内容。

## 文件清单

| 文件 | 用途 |
| --- | --- |
| `开发计划.md` | 核心设计文档（目标、边界、接口、架构） |
| `Backlog.md` | 产品待办列表，所有需求的唯一入口 |
| `任务清单.md` | 按 Sprint 分组的任务拆解与执行状态跟踪 |
| `变更记录.md` | 按 Sprint 分组的已完成变更归档记录 |

## 开发流程

```
用户提供需求 → 填充开发计划.md → 拆解到 Backlog.md → 按 Sprint 写入任务清单.md → 执行 → 回填变更记录.md
```
