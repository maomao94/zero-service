# 🚀 DJI 上云前置服务（Dji Gateway）开发计划 - 简化版

---

## 🧠 一、目标定义（核心）

构建一个轻量级 DJI 上云前置服务，用于：

> 👉 将业务系统请求（gRPC）转换为 DJI MQTT 协议指令，实现航线任务下发与 PSDK 控制能力

* 实现最小可用能力（MVP）

### ⛔ 设计前提（铁律）

> **S8 事故教训**：PM 对「平台能力接口」需求理解错误。以下规则为防止类似事故的刚性约束。

1. **SDK 严格对齐上云文档**：djisdk 包中的所有 Method 常量、Topic 定义、Protocol 结构体必须 1:1 来源于大疆上云 API 官方文档，**禁止自行发明 MQTT 方法或消息格式**
2. **Proto 分类优先按大疆文档**：所有对应大疆 Cloud API 标准 method 的 gRPC RPC，必须按大疆文档的官方分类归组（航线管理、远程调试、直播管理、指令飞行、相机/云台、固件管理、属性设置等），禁止自创分类
3. **平台能力接口的准确定义**：
   - **平台能力 = 大疆 Cloud API 中没有的、由本服务结合业务需求自定义或组合的接口**
   - 平台能力**不限于查询**，也可能是业务自定义的操作、组合指令、数据加工等
   - 只要对应大疆文档中已有的 method，就是协议转发，**不能归为平台能力**
   - 示例：`IsDeviceOnline`（大疆没有此 API，是我们根据 Status/OSD 上报数据自行维护的）= 平台能力 ✅
   - 反例：`ReturnSpecificHome`（大疆 Cloud API 标准 method: return_specific_home）= 协议转发 ❌ 不是平台能力
4. **上报数据先存后查**：hooks 收到的 OSD/State/HMS/Status 等设备上报数据，先持久化到内存缓存或存储，平台能力接口从缓存读取，而不是现查现发
5. **新增接口前检查清单**：
   - □ 该方法是否在大疆上云 API 文档中有定义？有→协议转发，归入对应大疆分类；没有→可能是平台能力
   - □ 数据流方向是什么？（云→设备 / 设备→云 / 本地查询 / 业务自定义组合）
   - □ proto 注释是否标注了对应的 DJI Cloud API 引用链接（协议转发类必须标注）？

# 🎯 二、核心能力

---

## 必须支持

### 1️⃣ 航线任务下发

* 支持业务传入 WPML URL
* 服务负责：

   * flighttask_prepare
   * flighttask_execute

---

### 2️⃣ PSDK 指令发送

* 支持透传 payload
* 下发至设备（psdk_write）

---

### 3️⃣ MQTT 协议适配

* 自动封装 topic
* 自动生成 tid
* 自动处理 ACK
* 使用 mqttx 包，参考 bridgemqtt

---

### 4️⃣ 基础响应能力

* 返回 tid
* 返回发送结果（成功 / 失败）

---

### 5️⃣ 简单日志能力

* 打印请求 / 响应日志
* 便于调试

---

# 🧩 三、系统定位
大疆前置的 api 服务，作为巡检系统调用使用

---

## ✅ 本系统是

* DJI 协议网关（Protocol Gateway）
* MQTT 指令转发器
* 业务系统的无人机控制入口

---

## ❌ 本系统不是

* 调度系统（不做任务编排）
* 工作流引擎（不做 DAG）
* 设备管理平台（不做完整设备模型）
* 实时流系统（不做全量事件订阅）

---

# ⚙️ 四、执行模型
grpc 转 mqtt 再用 antx 响应式模型拿数据 给 grpc 返回


---

## 调用方式

```text
业务系统 → gRPC → Gateway → MQTT → DJI设备
```

---

## 命令执行模型

* 单次请求 → 单次下发
* 等待 ACK 即返回
* 不跟踪完整任务生命周期

---

## 并发模型

* 多请求并发执行
* 每个请求独立处理

---

## 超时策略

* gRPC 控制整体超时
* MQTT ACK 设置等待超时（如 5s）

---

## 错误处理

* 发送失败立即返回
* ACK 超时返回失败

---

# 🌐 五、接口设计

## gRPC 服务定义

```proto
service DjiGatewayService {

  // ==================== 航线管理 ====================
  rpc ExecuteFlightTask(FlightTaskReq) returns (CommonRes);
  rpc CancelFlightTask(DeviceSnReq) returns (CommonRes);
  rpc PauseFlightTask(DeviceSnReq) returns (CommonRes);
  rpc ResumeFlightTask(DeviceSnReq) returns (CommonRes);
  rpc StopFlightTask(DeviceSnReq) returns (CommonRes);
  rpc ReturnHome(DeviceSnReq) returns (CommonRes);
  rpc ReturnHomeCancelAutoReturn(DeviceSnReq) returns (CommonRes);
  rpc ReturnSpecificHome(ReturnSpecificHomeReq) returns (CommonRes);

  // ==================== PSDK 透传 ====================
  rpc SendCustomDataToPsdk(CustomDataToPsdkReq) returns (CommonRes);

  // ==================== 指令飞行 / DRC ====================
  rpc DrcModeEnter(DrcModeEnterReq) returns (CommonRes);
  rpc DrcModeExit(DeviceSnReq) returns (CommonRes);
  rpc FlightAuthorityGrab / PayloadAuthorityGrab / TakeoffToPoint
  rpc FlyToPoint / FlyToPointStop
  rpc DroneEmergencyStop(DeviceSnReq) returns (CommonRes);
  rpc SendDrcCommand(DrcCommandReq) returns (CommonRes);

  // ==================== 远程调试（17 个） ====================
  rpc DebugModeOpen / Close / CoverOpen / CoverClose / CoverForceClose
  rpc DroneOpen / DroneClose / DeviceReboot / ChargeOpen / ChargeClose
  rpc DroneFormat / DeviceFormat / SupplementLightOpen / SupplementLightClose
  rpc BatteryStoreModeSwitchSwitch / AlarmStateSwitch / AirConditionerModeSwitch
  rpc BatteryMaintenanceSwitch(BatteryStoreModeReq) returns (CommonRes);

  // ==================== 相机 / 云台 ====================
  rpc CameraModeSwitch / CameraPhotoTake / CameraRecordingStart / CameraRecordingStop
  rpc CameraFocalLengthSet / GimbalReset / CameraAim(CameraAimReq)

  // ==================== 直播管理 ====================
  rpc LiveStartPush / LiveStopPush / LiveSetQuality
  rpc LiveLensChange(LiveLensChangeReq) returns (CommonRes);

  // ==================== 属性设置 ====================
  rpc SetProperty(SetPropertyReq) returns (CommonRes);

  // ==================== 固件管理 ====================
  rpc OtaCreate(OtaCreateReq) returns (CommonRes);

  // ==================== 平台能力接口（非大疆标准 API，本服务自定义） ====================
  // 平台能力 = 大疆 Cloud API 中没有的、由本服务结合业务需求自定义或组合的接口。
  rpc IsDeviceOnline(DeviceSnReq) returns (DeviceOnlineRes);
  // TODO: 后续按业务需求扩展（如 OSD 状态查询、HMS 告警查询等）
}
```

> 完整 Proto 定义见 `app/djigateway/djigateway.proto`。
> 当前共计约 50 个 RPC 方法。
