# dtui 面板交互优化 + 配置表单化 + 发布包配置

## Goal

提升 dtui TUI 的交互体验：配置编辑改用表单、新增发布包快捷部署、所有面板自适应终端尺寸。

## Requirements

### R1: 配置表单化

设置页的新增/编辑操作从 exec 面板纯文本输入改为弹窗表单：
- 新增编排目录：字段（名称、路径），placeholder 提示格式
- 新增前端发布目标：字段（名称、容器名、HTML 路径、备份目录），带默认值
- 编辑已有条目：回填当前值到表单
- 表单组件复用 Panel 接口，支持 Tab 切换字段、Enter 提交

### R2: 发布包配置

新增配置类型 `deploy_packages`：
```json
{
  "deploy_packages": [
    {"name": "dist.zip", "path": "/path/to/dist.zip"},
    {"name": "前端产物", "path": "/path/to/build/folder"}
  ]
}
```
- 前端发布视图显示已配置的发布包列表
- 选中后按 `d` 直接部署，无需手输路径
- 支持 zip 和文件夹两种类型
- 设置页支持新增/编辑/删除发布包

### R3: 面板自适应

所有面板根据终端尺寸自适应布局：
- 日志面板：长日志行根据终端宽度自动换行（不截断）
- 操作记录面板：更大展示区域，信息密度优化
- exec 面板：输入区域和输出区域根据终端高度动态分配
- inspect/stats/镜像历史面板：内容区域自适应

### R4: 日志换行

容器日志面板支持自动换行：
- 超过终端宽度的日志行自动换行显示
- 保持行号对齐
- 搜索高亮在换行后仍然正确

## Acceptance Criteria

- [ ] 设置页新增编排目录用表单，带字段标签和 placeholder
- [ ] 设置页新增前端发布目标用表单，带默认值
- [ ] 编辑已有条目回填当前值
- [ ] `deploy_packages` 配置支持，在前端发布视图中显示
- [ ] 选中发布包按 `d` 直接部署
- [ ] 日志长行自动换行
- [ ] 操作记录面板展示区域增大
- [ ] exec 面板输入/输出动态分配
- [ ] 所有面板在 80x24 和宽屏终端下正常显示
- [ ] build + vet + test 全通过

## Constraints

- 遵循 `.trellis/spec/backend/dtui-conventions.md`
- Panel 接口不变
- 配置格式向后兼容（新字段可选）
