# 版本模板说明

本目录提供新版本开发计划的标准模板文件。
当需要开始新版本开发时，复制本目录下的模板文件到新版本目录中。

## 使用方法

```bash
# 示例：创建 0.0.3 版本目录
cp -r CP-开发计划/template/ CP-开发计划/0.0.3/
```

然后修改各文件中的 `{版本号}` 占位符为实际版本号。

## 文件清单

| 文件 | 用途 |
| -- | -- |
| `开发计划.md` | 核心设计文档模板（目标、边界、接口、架构） |
| `任务清单.md` | 任务拆解与执行状态跟踪模板 |
| `变更记录.md` | 变更记录模板（按轮次倒序） |
| `下一步计划.md` | 用户草稿输入模板 |
