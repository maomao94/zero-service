# Calculator Skill

## 概述

Calculator Skill 提供数学计算能力，支持加减乘除、幂运算等基本数学运算。

## 何时使用

当用户需要计算数学表达式时使用此技能，例如：
- 简单的算术计算 (2+3*4)
- 百分比计算
- 幂运算 (2^10)
- 复杂的嵌套表达式

## 使用方法

调用 `calculator` 工具，输入数学表达式。

### 参数

- `expression`: 数学表达式（字符串）
  - 支持操作符: `+`, `-`, `*`, `/`, `^`
  - 支持括号: `(`, `)`
  - 支持小数: `3.14`

### 示例

```
输入: 2 + 3 * 4
输出: 14

输入: (10 + 5) / 3
输出: 5

输入: 2 ^ 10
输出: 1024
```

## 限制

- 不支持三角函数、对数等高级数学运算
- 不支持变量
- 除法除以零会返回错误

## 示例对话

**用户**: 计算 25% 的 200 是多少？

**助手**: 让我用计算器来算：
- 25% = 0.25
- 0.25 * 200 = 50

所以 25% 的 200 是 **50**。

---

**用户**: 帮我算一下 2 的 10 次方

**助手**: 使用幂运算计算：
- 2^10 = 1024

结果是 **1024**。
