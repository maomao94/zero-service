使用说明

接口说明

  • 提交任务: POST /async/tool/call - 提交异步工具调用任务
  • 查询结果: GET /async/tool/result/{task_id} - 轮询查询任务状态和结果

请求示例

{
  "server": "mcpserver",
  "tool": "test_progress",
  "args": {"steps": 5, "interval": 200}
}

响应示例

{
  "task_id": "uuid-xxx",
  "status": "completed",
  "progress": 100,
  "messages": [
    {"progress": 0, "message": "开始执行...", "time": 1711234567},
    {"progress": 33.33, "message": "步骤 1/3", "time": 1711234568},
    {"progress": 100, "message": "执行完成", "time": 1711234570}
  ],
  "result": "{\"success\": true, \"data\": \"...\"}"
}

消息说明

  • 前端根据进度推断显示：▶ 开始（0%）→ 进度更新 → ✓ 完成（100%）
  • 进度范围 0.0~100.0
